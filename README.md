## 项目介绍：

仿照小兔鲜儿网站，练习vue3相关技术栈

## 项目结构：

|--	api	接口

​		|--  index.ts  针对axios的二次封装，设置拦截器等

​		|--  api-rabbit.ts  采用小兔鲜儿网站接口，获取相关数据

|--  assets	静态资源

​		|--  iconfont  项目采用图文字的资源文件

​		|--  images  项目图片资源文件

|--  components	所有组件

​		|--  custom  项目整体公共组件

​		|--  home 页面组件，公用性不够强

|--  router	路由

|--  store	状态管理，pinia

|--  utils	公共文件

​		|--  tools.ts	公共处理函数文件

|--  views	视图





# 技术栈：

#### vue3+vite+ts+pinia+axios
